
## Скрипты

Установить сборку
```
npm run create
```

Поднять дев для разработки
```
npm run start
```

Собрать проект
```
npm run build
```

Запустить деплой
```
npm run deploy
```

Сгенерировать шаблон страницы
```
npm run gen:layout
```

Сгенерировать страницу
```
npm run gen:page
```

Сгенерировать блок/компонент/модуль
```
npm run gen:block
```
